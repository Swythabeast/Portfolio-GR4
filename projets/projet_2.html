Système de réservation

Le projet **Système de réservation** est une plateforme web développée pour permettre aux utilisateurs de réserver des services ou des ressources en ligne. Cette application offre une interface intuitive pour la gestion des réservations, avec des fonctionnalités telles que :

  • Création et gestion des comptes utilisateurs.
  • Gestion des disponibilités et des créneaux horaires.
  • Notifications par e-mail et SMS pour les confirmations et rappels.
  • Intégration avec des calendriers (Google Calendar, Outlook).

Technologies utilisées

Ce projet a été développé en utilisant les technologies suivantes :

  • Front-end : HTML, CSS, JavaScript, Framework Bootstrap.
  • Back-end : Node.js, Express.js.
  • Base de données : MongoDB pour la gestion des réservations et des utilisateurs.
  • API : Intégration d'une API pour l'envoi d'e-mails et SMS.
