Gestion des Notes

Ce projet est une application web permettant aux utilisateurs de gérer leurs notes personnelles. Elle inclut des fonctionnalités de création, modification, suppression, et organisation des notes dans des catégories.

Fonctionnalités principales :

  • Créer, modifier, et supprimer des notes.
  • Organisation des notes par catégories ou tags.
  • Recherche rapide des notes avec des filtres.
  • Authentification sécurisée via JWT.
  • Sauvegarde et cryptage des données dans MongoDB.

Technologies utilisées :

  • Frontend : React.js avec une interface utilisateur moderne et réactive.
  • Backend : Node.js avec Express pour gérer les API REST.
  • Base de données : MongoDB pour le stockage des données, avec chiffrement AES pour sécuriser les informations.
  • Sécurité : Authentification et gestion sécurisée des sessions avec JWT, protection contre les attaques XSS et CSRF.
  • CI/CD : Pipeline GitLab CI intégrant des tests de sécurité automatisés (OWASP ZAP).

Compétences démontrées :

  • Création d'interfaces utilisateur modernes et intuitives.
  • Mise en œuvre de pratiques de sécurité avancées pour les API et les données.
  • Intégration de pipelines CI/CD sécurisés pour garantir une livraison rapide et fiable.
  • Gestion de projet full stack, de la conception à la mise en production.
